# Supported Formats

Media Writer 的输入处理原则是：**能直接读取就直接读取；不能直接读取就先转成 Markdown/TXT；再进入播客或微信草稿流程**。

## 已支持或推荐路径

| 格式 / 来源 | 播客分支 | 微信草稿分支 | 说明 |
| --- | --- | --- | --- |
| Markdown `.md` | 支持 | 支持 | 最推荐格式 |
| TXT `.txt` | 支持 | 支持 | 适合转录稿和纯文本资料 |
| 微信公众号文章 | 支持 | 支持 | 需要先抓取或导出正文 |
| 普通网页 | 支持 | 支持 | 优先直接 URL；失败时抓取正文 |
| X / Twitter | 支持 | 支持 | 可受登录和反爬限制影响 |
| YouTube | 支持 | 需转文本 | 播客分支推荐直接添加 URL 到 NotebookLM |
| 小宇宙 / 喜马拉雅 | 需转录稿 | 需转录稿 | 可使用外部转录或脚本辅助 |
| Bilibili | 需转录稿 | 需转录稿 | 视视频字幕/转录可用性而定 |
| PDF | 需转文本 | 需转文本 | 先用本地工具转 Markdown/TXT |
| DOCX / PPTX | 需转文本 | 需转文本 | 先提取正文 |
| EPUB | 需转文本 | 需转文本 | 适合长文阅读资料 |

## 暂不承诺的能力

- 绕过付费墙或权限限制。
- 自动把 MP3 嵌入微信公众号草稿。
- 自动生成 PPT、思维导图或 Quiz。
- 对所有网页进行 100% 成功抓取。

这些能力如果后续实现，会进入 Roadmap 或 Changelog。

## 输出格式

播客分支：

- MP3 音频
- 标题简介 TXT
- JPG 封面

微信草稿分支：

- 改写 Markdown
- WeChat HTML
- 图片映射 JSON
- 微信草稿 JSON
- 草稿上传结果 JSON
