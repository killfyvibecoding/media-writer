# Changelog

所有重要变更都会记录在这里。

## [0.2.0] - 2026-06-02

### Added

- 增加内容诊断前置层，输出 `*_content_diagnosis.md` 和 `*_content_diagnosis.json`。
- 增加 Xiaohongshu 科普帖生成脚本和测试。
- 增加 NotebookLM PPT、思维导图和 artifact 流程说明。
- 增加 PPT+播客 HyperFrames 视频流程说明和测试。
- 增加 NotebookLM podcast 后台运行、runtime state/log 隔离和 `.m4a` 交付说明。
- 增加 PPT 设计提示词模块，约束高级极简初创企业视觉风格。

### Changed

- README、Quick Start、FAQ、Supported Formats、Demo 全面更新为当前组合输出池流程。
- GitHub Issue / PR 模板增加诊断、PPT、思维导图、小红书和视频分支检查项。
- 播客流程改为诊断驱动的 NotebookLM instructions、标题简介和封面方向。
